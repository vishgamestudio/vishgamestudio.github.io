Privacy Policy

Introduction

This page defines how your private data is being handled by us here at VISH GAME STUDIO ("we" or "us") when you ("you") are playing Just King (the "Game").

The Privacy Policy was last updated on the 7th of November 2024.

Policy changes

We may change these Terms and will notify users inside the Game.

Required Data

In order to provide the Game to you, we need to collect certain data about you. Which data and for what purpose is defined below.

User Provided Data

The following data is provided by you and stored on our servers:

UserID (Applies to all platforms): Required for sign-up.

Automatic Data Collection

The following data is collected and stored automatically when playing the Game.

Steam User ID (Applies only to Steam): Used to allow purchases through the Steam store, and to provide customer support.

User Generated Data

While playing the Game, you generate data that we need to process in order to provide the Game to you, such as what matches you have played, what parties have you made, which items you own, etc.

Some of this generated data will be publicly available to other users. Things like parties you have built, ranks that you have attained on the leaderboards, etc.

We store and process this data to provide the Game to you.

Legitimate interest

We process user generated data in order to:

Decide future development work / Balance patches

Evaluate game performance

For data solely used for analytics, it will be retained for no more than 1 month.

Third Party Data Privacy

The Game uses third-party services that may collect information used to identify you:

Unity (Applies to all platforms): The Game is made in the Unity game engine and uses the Unity Game Services for collection and storing crash and exception reports. The service collects unique installation-specific ID, device name, operating system version, Game configuration at the time of the error, usage time and date, and other statistics.

Please read their privacy policy.

Security

All data transmissions use encryption, and are stored in places that use access control. However, while we strive to protect your personal information, we cannot guarantee absolute security due to the inherent vulnerabilities of internet-based transmissions.

Links to Other Sites

The Game contains links to other sites not operated by us. Please use at your own discretion and be sure to review their privacy policies.
